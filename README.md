# Instrucciones de Uso:

## Paso 1:
### Se abrirá una ventana como la siguiente pidiendo que ingrese un elemento al árbol, debe digitarlo y presionar el boton Aceptar:

![avl3](https://user-images.githubusercontent.com/55126792/110189093-92ddf500-7de3-11eb-98aa-6f6259bc2994.PNG)

## Paso 2:
### Si se desea seguir ingresando numeros al arbol precione Aceptar en la siguiente ventana que aparece

![avl4](https://user-images.githubusercontent.com/55126792/110189152-c456c080-7de3-11eb-9db8-20460196cf1d.PNG)


## Paso 3:
### Si selecciona que no desea ingresar mas elementos aparecera la siguiente ventana con el arbol 

![avl5](https://user-images.githubusercontent.com/55126792/110190073-2cf36c80-7de7-11eb-803b-e9ed307d0454.PNG)


## Paso 4:
### Para salir del programa se debe presionar el botón salir, y se mostrara un mensaje antes de finalizar el programa:
